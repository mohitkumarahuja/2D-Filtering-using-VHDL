To Run the VHDL Code, Run Mohit_VHDL_Project.xise

To run the Matlab Code, Just add the folder directory to matlab and run Project_Mohit.m

This Folder Consists:

1. FILTER_MOHIT_Process.vhd	 --> Main Processing Unit.
2. Cache_Memory.vhd	 	 --> Cache Memory Unit.
3. Mohit_TB.vhd			 --> Test Bench of the code.
4. Project_Mohit.m		 --> Used to print the Output in Matlab.
5. Report of the project
6. Problem Statement of the project
